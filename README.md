## Getting Started

### Prerequisites

Ensure you have the following installed:

- Web browser (e.g., Chrome, Firefox)

### Installation

Clone the repository:

```bash
git clone https://github.com/sign-maker/psychologists-services.git
cd psychologists-services


npm install
npm run dev
```
